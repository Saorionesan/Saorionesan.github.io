角色：Java开发

技术栈：SpringBoot + SpringCloudAlibaba + Mybatis + MySQL + Redis + MongoDB + RocketMQ + Jenkins + Docker + K8S

负责模块：

  • ipaas-gateway：微服务网关实例，提供请求路由、负载均衡、认证鉴权、监控日志等能力
  • ipaas-dfa-manager：节点网服务管理调度系统，负责服务的管理、部署、监控和运行时控制

项目成果：

  • 支持单个节点网服务100+任务的执行与排队
  • 通过任务日志回放功能快速定位错误节点
  • 节点网服务应用于金融、制造等多个行业

演示地址： https://ipaas.lab.clickpaas.com
